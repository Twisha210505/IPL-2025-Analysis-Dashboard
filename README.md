# 🏏 IPL 2025 Performance Dashboard (Power BI)

An interactive **Power BI dashboard** analyzing the complete **IPL 2025 season performance** using batting, bowling, and match-level datasets.

This dashboard provides insights into:

- **Batting Performance Analysis**
- **Bowling Performance Analysis**
- **Match-wise Results**
- **Player Comparison Metrics**
- **Team Performance Overview**

---

# 📊 Dashboard Preview

This project contains:

### **Batting Dashboard**
- Total Runs Scored
- Strike Rate Analysis
- Average by Player
- Boundary Contribution (%)
- Runs by Match
- Fours & Sixes Distribution

### **Bowling Dashboard**
- Total Wickets
- Economy Rate Analysis
- Bowling Average
- Wickets by Player
- Runs Conceded by Match
- Match vs Wickets Trends

### **Match Dashboard**
- Match Results
- Winning Margin Analysis
- Venue-wise Performance
- Team-wise Results Distribution

# 📈 Key Insights

### Batting
- Sai Sudharsan led the run charts.
- High strike-rate consistency from top-order batters.
- Boundary contribution heavily influenced match outcomes.

### Bowling
- Prasidh Krishna was the highest wicket-taker.
- Economy control played a key role in match-winning spells.

### Team Trends
- Gujarat Titans showed dominance in both batting and bowling.
- RCB secured important close wins under pressure.
